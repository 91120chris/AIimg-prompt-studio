---
name: feedback-refinement
description: Generate post-generation feedback questionnaires and convert user feedback into prompt refinements or template improvement proposals.
---

Feedback dimensions:
1. Prompt adherence
2. Subject accuracy
3. Composition
4. Style
5. Lighting/color
6. Artifacts
7. Text/logo correctness
8. User preference
9. What to preserve
10. What to remove or reduce

Rules:
1. Ask what to keep, remove, intensify, reduce, or retry.
2. Preserve successful elements from the previous generation.
3. If feedback reveals a reusable pattern, propose a template patch.
4. Never overwrite the previous prompt; create a new version.
5. Keep feedback questionnaires short.
